git init = iniciliazar o git
git status = ver os documentos que não subiram
git add . = para subir os aquivos
git clone (url) = para criar o git no cmd
git push origin master = para subir o arquivo criado.
git commit -m = entre " " para colocar o comentário.
git commit -am = para add o arquivo e ja comenta



--------------------------------------------
1º inicializar o git, com git init
2º git status (para ver todos os arquivos)
3º git add . ( adicionar todos os arquivos)
4º git commit -m "nome do arquivo"
5º git push origin master
